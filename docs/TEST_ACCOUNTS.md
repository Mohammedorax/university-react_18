# حسابات التجربة (Mock API)

يستخدم التطبيق في التطوير والنشر التجريبي **`mockApi`** داخل المتصفح. يمكن تسجيل الدخول بالحسابات التالية (راجع أيضًا `src/services/mockApi/auth.ts` إن تغيّرت القيم).

| الدور | البريد | كلمة المرور |
|--------|--------|-------------|
| مدير | `admin@university.com` | `admin123` |
| مدرس | `teacher@university.com` | `teacher123` |
| طالب | `student@university.com` | `student123` |

## ملاحظات

- البيانات محاكاة في الذاكرة/التخزين المحلي حسب تنفيذ `mockApi`.
- عند ربط **PostgreSQL** والباكند الحقيقي، تُستبدل هذه الحسابات بمصادقة السيرفر.
