# Supabase Free Tier - PostgreSQL Database
# Steps to get your DATABASE_URL:

# 1. Go to: https://supabase.com
# 2. Create free account
# 3. Click "New Project"
# 4. Fill project details:
#    - Name: university-db
#    - Password: generate strong password
#    - Region: choose closest to you

# 5. Wait for project to be ready (~2 minutes)

# 6. Go to Settings > Database
# 7. Copy "Connection string" (password included)

# 8. Replace below with your actual connection string:
# DATABASE_URL="postgresql://postgres:your_password@your-project-ref.supabase.co:5432/postgres"

# 9. Also get these from Settings > API:
#    - PROJECT_URL="https://your-project-ref.supabase.co"
#    - ANON_KEY="your-anon-key"
#    - SERVICE_ROLE_KEY="your-service-role-key"

# Note: Supabase free tier includes:
# - 500MB Database
# - 1GB File Storage
# - 50MB Edge Functions
# - 2GB Bandwidth/month
